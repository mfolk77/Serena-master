## Screenshot Guidelines

### Technical Requirements
- **Format**: PNG (preferred) or JPEG
- **Color Space**: sRGB
- **Resolution**: Retina quality (2x)
- **Compression**: Minimal, maintain quality

### Content Guidelines
- Use realistic, helpful conversations
- Show the app in its best light
- Ensure text is readable at all sizes
- No personal information visible
- Consistent UI theme throughout
- Professional appearance

### Naming Convention
- `01_main_interface.png`
- `02_voice_input.png`
- `03_settings_privacy.png`
- `04_conversations.png`
- `05_help_onboarding.png`

## Creation Process

1. **Setup Clean Environment**
   - Fresh macOS installation or clean user account
   - Remove personal files and information
   - Set consistent theme (light/dark)
   - Ensure proper window sizing

2. **Prepare Sample Content**
   - Create realistic conversations
   - Use professional, helpful examples
   - Avoid controversial or sensitive topics
   - Show AI capabilities clearly

3. **Capture Screenshots**
   - Use built-in Screenshot app (Cmd+Shift+4)
   - Ensure proper resolution and quality
   - Capture at exact required dimensions
   - Take multiple versions for selection

4. **Post-Processing**
   - Crop to exact dimensions
   - Adjust brightness/contrast if needed
   - Remove any artifacts or imperfections
   - Optimize file size without quality loss

## Quality Checklist

- [ ] All text is clearly readable
- [ ] No personal information visible
- [ ] Consistent theme and appearance
- [ ] Professional conversation examples
- [ ] Proper resolution and dimensions
- [ ] No UI glitches or artifacts
- [ ] Features are clearly demonstrated
- [ ] Privacy aspects are highlighted

## App Store Connect Upload

1. Log into App Store Connect
2. Navigate to your app's product page
3. Go to App Store tab
4. Upload screenshots in order
5. Add localized descriptions for each
6. Preview how they appear in the store
7. Save changes and submit for review

## Localization

For international markets, consider:
- Translating UI text in screenshots
- Cultural appropriateness of content
- Local privacy law compliance
- Regional design preferences

Screenshots should be created for each supported locale if UI text is visible.
